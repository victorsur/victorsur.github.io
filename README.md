# victorsur.github.io

Portfolio personal para GitHub Pages.

## Estructura base

- `index.html`: pagina principal (hero, sobre mi, proyectos, contacto).
- `styles.css`: estilos visuales, responsive y tema claro/oscuro.
- `main.js`: interacciones (menu movil, reveal, back-to-top, tema, anio).
- `wordle/`: mini juego Wordle en espanol con HTML/CSS/JS.
- `Curriculum2026io.pdf`: curriculum descargable.

## Ejecutar en local

Puedes abrir `index.html` directamente en el navegador, o servir con Python:

```bash
cd /home/victor/Web/github-page/victorsur.github.io
python3 -m http.server 8080
```

Luego abre `http://localhost:8080`.
